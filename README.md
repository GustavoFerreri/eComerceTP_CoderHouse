# Aplicacion Comercio Electronico

El proyecto recrea la acumulacion de conceptos aprendidos en el curso de coderhouse, desde la instalacion, hasta el trabajo en base de datos.

[![Build Status](https://travis-ci.org/joemccann/dillinger.svg?branch=master)](https://github.com/GustavoFerreri/eComerceTP_CoderHouse)

Para clonar el el proyecto  dejo debajo una ayuda
https://docs.github.com/es/repositories/creating-and-managing-repositories/cloning-a-repository

cabe aclarar que utiliza variables de entorno, preservando los datos sensibles de conexion.

## Para correr correctamente
En el directorio del proyecto, ejecutamos la siguiente linea de comando
### `npm start`
